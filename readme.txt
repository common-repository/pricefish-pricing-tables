== Installation ==

= Creating a pricing table or page =

1. Go to PriceFish and start our fast generator [PriceFish](http://www.pricefish.com).
2. Select designs
3. Enter details
4. Click Embed on PriceFish Dashboard
5. Click the "Publish Table" button, and look for the "Table ID" section. Copy it, and write it somewhere (we will use it later).

= Attach a pricing table to a Wordpress post =

1. Create a new post and look for the PriceFish button button in your editor (a little fish with $ sign)
2. A popup will appear, enter the relevant project id (you can copy it form pricefish dashboard , just click on embed).
3. If you can't find the "Pricing Table" button, you may also write [pricefish projectid="your project id(a guid)"] in the editor, and replace "your project id (a guid)" with the relevant project id.

Arguments provides by the shortcode:

* 'projected' Your project ID from PrieFish
